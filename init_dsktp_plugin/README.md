# init_dsktp_plugin

How to create a Flutter Desktop Plugin

## Installation

`pubspec.yaml`

```pubspec.yaml
...
dependencies:
  init_dsktp_plugin:
    git:
      url: https://github.com/AseemWangoo/plugins.git
      path: ./init_dsktp_plugin
...
```

## Usage

```dart
import 'package:init_dsktp_plugin/init_dsktp_plugin.dart';

InitDsktpPlugin.platformVersion
```
